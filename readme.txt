# 币安BTC量化交易数据分析项目

## 项目简介

本项目用于收集、分析币安（Binance）BTC交易数据，并通过技术指标挖掘量化交易因子。
目标：发现具有统计显著性的价格预测因子，为量化交易策略提供数据支持。

## 目录结构

### 核心文件说明

- **source.txt** - 可用指标清单和计算方法
  - 包含40+技术指标的详细计算公式
  - 数据字段说明和时间范围
  - 指标组合建议

- **result.txt** - 因子挖掘结果记录
  - 格式要求：样本集/测试集时间范围、样本数、成功率
  - 当前已验证因子：
    * 连续下跌8-9根K线后反弹概率60%
    * 连续上涨7-9次后回调概率55%+
    * 资金费率与5分钟K线涨跌相关性

- **readme.txt** - 本文件，项目说明文档

### 数据采集脚本

| 脚本文件 | 功能说明 | 数据频率 |
|---------|---------|---------|
| get_klines.py | K线数据（OHLCV） | 5分钟 |
| get_funding_rate.py | 资金费率历史 | 8小时 |
| get_open_interest.py | 合约持仓量 | 5分钟 |
| get_long_short_ratio.py | 多空持仓人数比 | 5分钟 |
| get_taker_buy_sell_ratio.py | 主动买卖量 | 5分钟 |
| get_top_long_short_account_ratio.py | 大户账户数多空比 | 5分钟 |
| get_top_long_short_ratio.py | 大户持仓量多空比 | 5分钟 |
| get_basis.py | 基差数据 | 待定 |

### 数据文件

所有CSV数据文件存放在项目根目录，命名格式：
- K线: BTCUSDT_5m_起始日期_结束日期.csv
- 其他: BTC[指标名称]历史.csv

## 使用流程

### 1. 数据采集
```bash
# 运行相应的Python脚本采集最新数据
python get_klines.py          # 获取最新K线
python get_funding_rate.py    # 获取资金费率
# ... 其他数据采集脚本
```

### 2. 因子研究
- 查阅source.txt了解可用指标和计算方法
- 选择指标组合进行回测
- 建议组合：
  * 趋势类：MACD + EMA + ADX
  * 合约专属：资金费率 + 持仓量 + 多空比

### 3. 回测验证
- 划分训练集和测试集（建议7:3比例）
- 记录样本数量、时间范围
- 计算成功率、夏普比率等指标

### 4. 结果记录
将验证通过的因子写入result.txt，格式示例：
```
连续下跌8-9根k线后，下一根k线上涨概率接近60%
[样本集: 2024-08-06~2025-12-31, 样本数: 1234]
[测试集: 2026-01-01~2026-04-09, 成功率: 58.7%]
```

## 技术指标分类

### 趋势类 (7种)
SMA, EMA, MACD, 布林带, SAR, 一目均衡表, ADX

### 震荡类 (6种)
RSI, Stochastic, KDJ, CCI, Williams %R, ROC

### 成交量类 (5种)
OBV, MFI, VWAP, Volume MA, CMF

### 波动性类 (4种)
ATR, 历史波动率, Keltner Channel, Donchian Channel

### 合约专属 (5种)
持仓量变化率, 资金费率, OI/Volume比, 多空比背离, 主动买卖压力指数

详细计算公式请查看source.txt

## 已验证因子总结

### 1. 连续下跌反弹因子
**规则**: 连续下跌8-9根K线后，下一根K线上涨概率60%
**适用场景**: 5分钟级别短线交易
**风险**: 需配合支撑位和成交量确认

### 2. 连续上涨回调因子
**规则**: 连续上涨7-9次后，下一根K线下跌概率55%+
**适用场景**: 5分钟级别短线交易
**风险**: 强趋势市场可能失效

### 3. 资金费率预测因子
**规则**:
- 资金费率下降 → 5分钟K线上涨概率更高
- 资金费率升高 → 5分钟K线下跌概率更高
**适用场景**: 合约交易，配合持仓量分析
**机制**: 资金费率反映市场情绪和多空力量对比

## 注意事项

1. **数据时差**: 所有时间戳为UTC时间，需转换为本地时区
2. **数据完整性**: 定期检查数据缺失，避免影响回测结果
3. **过拟合风险**: 避免在同一数据集上反复优化参数
4. **交易成本**: 回测时务必考虑手续费和滑点
5. **市场环境**: 牛熊市、高低波动环境下因子表现可能差异巨大
